# Password Generator (Python)

## Description
This is a Python-based **Password Generator** that creates strong and customizable passwords based on user preferences.  
The user can specify:
- Minimum password length
- Whether to include numbers
- Whether to include special characters

The program ensures that the generated password meets all selected criteria and is at least the minimum length provided by the user.

This project is ideal for beginners learning:
- Python basics
- Loops and conditionals
- Functions
- String handling
- Randomization

## How It Works
1. The user inputs the minimum password length.
2. The user chooses whether to include numbers and special characters.
3. The program randomly generates characters until:
   - The minimum length is met
   - All selected criteria are satisfied
4. The final password is displayed to the user.
